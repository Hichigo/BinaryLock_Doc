# Introduction

## Description

Combination lock based on binary input. Contains 6 pre-configured castles of varying difficulty. Customized locks can be hung on doors or walls, and you can also make something unusual out of them, like stone buttons in a cave or a device for converting binary digits to decimal numbers. For locks, there is a functionality with the output of numbers or with a light indicator.
